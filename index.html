Validates: routing + traffic isolation tests

Dynamic Routing Design Using OSPF for Enterprise Scalability

Three-router topology with multiple LANs running OSPF, including adjacency verification. A high-signal skill for junior network roles.

Validates: neighbor adjacency + route tables
